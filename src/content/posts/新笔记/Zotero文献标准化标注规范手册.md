## 1 核心原则：固定颜色 + 统一标签 + 结构化笔记

建立**永久不变的六色标注体系**，统一视觉逻辑、检索标签、笔记模板、阅读工作流，

做到：一眼识别文本类型、可快速检索、可批量复用、写作直接溯源，杜绝乱标、重复标记、后期找不到重点。

## 2 六色标准化颜色体系

### 2.1 颜色总表（顺序固定｜标识｜定义｜适用场景）

| 颜色    | 图里的标识            | 用途定义                           |
| :---- | :--------------- | :----------------------------- |
| 🟨 黄色 | `general yellow` | 背景 / 核心论点 / 创新点 / 研究目的/ （What） |
| 🟩 绿色 | `general green`  | 关键结果 / 数据 / 图表结论（Results）      |
| 🟦 蓝色 | `general blue`   | 方法 / 实验 / 技术细节（Methods）        |
| 🟧 橙色 | `general orange` | 待验证 / 疑问 / 待查（Question）        |
| 🟪 紫色 | `general purple` | 与我的研究关联 / 可引用 / 灵感（Relevance）  |
| 🟥 红色 | `general red`    | 缺陷 / 局限 / 批判（Critique）         |

### 2.2 精简速记规则

- 🟨 黄：**为什么做**（目的、空白、创新）
- 🟩 绿：**做出了什么**（数据、结果、现象）
- 🟦 蓝：**怎么做的**（数据、软件、流程、分组）
- 🟧 橙：**看不懂、有疑问**（待查、存疑）
- 🟪 紫：**对我有用、可引用**
- 🟥 红：**文章不足、局限性**

### 2.3 颜色设计逻辑（标准化依据）

1. **🟨 黄色｜核心论点**
    黄色视觉优先级最高，黑白 PDF 中最醒目，快速定位全文研究初衷与科学创新，适合摘要、引言、假设部分标记。
2. **🟩 绿色｜客观结果**
    绿色象征稳定、可信、已验证，专门标记量化数据、差异基因、通路结论、实验表型等客观产出。
3. **🟦 蓝色｜实验方法**
    蓝色代表理性、逻辑、流程化，统一标记数据库、R 包、分析流程、实验分组、统计方法，全部为可复现内容。
4. **🟧 橙色｜疑问待查**
    橙色介于提醒与中性之间，用于标记模糊表述、机制不清、结论矛盾、需要后续文献补充验证的内容。
5. **🟪 紫色｜课题关联**
    紫色为专属关联色，筛选时快速区分「通用文献内容」和「本人课题可复用内容」，写综述 / 论文高效溯源。
6. **🟥 红色｜局限批判**
    红色为风险警示色，统一标注研究短板、样本缺陷、结论限制、无法解释的实验矛盾，引用时规避学术漏洞。

## 4 Zotero 7 PDF 标注工具区分与使用规范

### 4.1 六大标注工具功能差异

1. **高亮文本（A 高亮图标）✅ 主力常用**
    给文字添加底色高亮，六色全部使用该工具，是全文最核心标记方式。
2. **下划线文本** ⚙️ 辅助补充
    用于疑问内容二次标注，搭配橙色使用，适合划记矛盾、模糊语句。
3. **便签注释** 📝 深度批注
    不遮挡原文，适合写长思考、机制解读、个人理解，Better Notes 可正常识别。
4. **文本框注释（T）** ⚠️ 少用
    直接覆盖 PDF 原文，破坏排版，仅适合临时极简关键词备注。
5. **矩形框选** 📊 图表专用
    框选图、表、通路图、模型图，搭配绿色（结果）、蓝色（方法）使用。
6. **手绘涂鸦** ✏️ 临时草稿
    快速圈画、箭头标注、简易思路梳理，不做长期标准化标注。

### 4.2 工具使用原则

- 文字内容：只用「**高亮 + 评论批注**」
- 图片 / 表格：只用「**矩形框选 + 注释**」
- 疑问补充：下划线 + 橙色双重标记
- 禁止乱涂乱画、滥用文本框，保证 PDF 整洁统一

## 5 Zotero 高效快捷键（固定记忆）

- `Alt+1`：快速切换高亮工具
- `1 / 2 / 3 / 4 / 5 / 6`：一键切换 黄 / 绿 / 蓝 / 橙 / 紫 / 红
- `Ctrl + [/]`：上一条 / 下一条标注
- `Ctrl + L`：快速打开左侧注释列表
- 选中文字直接批注，全程无多余点击

## 6 Better Notes 结构化笔记规范

### 6.1 模板适配说明

配套专属六色分类模板，色值与 Zotero 7 内置高亮**完全匹配**，

一键提取注释后自动按颜色分栏：

- 黄色区：核心观点
- 蓝色区：实验方法
- 绿色区：研究结果
- 橙色区：疑问待查
- 紫色区：课题关联
- 红色区：研究局限

## 7 标准阅读工作流（初读 → 精读 → 整理 → 复用）

### 7.1 初读泛读（5–10min）

只标：**黄色（目的 / 创新）+ 绿色（核心结果）**

快速判断文献价值，添加标签：`#初读`

### 7.2 全文精读（30–60min）

严格按照六色体系全覆盖标记，

**每一条高亮必须附带简短批注**，不做无意义纯高亮。

### 7.3 即时整理（5min）

使用 Better Notes 一键生成结构化笔记，

补充个人总结、机制梳理、可复用要点。

### 7.4 长期复用

写作 / 综述阶段：按「颜色 + 标签」检索，

点击笔记内 `zotero://` 链接，一键跳转原文定位。

## 8 硬性避坑规范（长期遵守）

1. 禁止随意新增、修改、替换 Zotero 原生 6 色，保证模板永久兼容
2. 拒绝全篇高亮，只标记**关键句子、核心短语**
3. 方法与结果严格拆分：「怎么做 = 蓝，得到什么 = 绿」
4. 所有标注附带简短文字批注，拒绝 “只标不写”
5. 定期备份 Zotero 数据 + PDF 标注，防止标注丢失
6. 图表、公式统一用框选工具，不强行文字高亮
